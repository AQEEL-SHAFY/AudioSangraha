AudioSangraha - An Approach for Transforming Sinhala Audios into Summaries 

This is a system for transforming Sinhala language audio data into summaries. 
The continuous of audio files are taken as input and produce the text output as a paragraph. For converting the audio into text, the author has used Whisper AI for Sinhala language. For training the model author has used a custom dataset. And then using the extractive summarization method, the converted text will be produce as a summarized version. Finally, the user will be able to get a summarized version of audio unless listening to continuous of audios.
